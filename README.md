## 1. 프로젝트 개요

* **기획 배경**

   - 일반적인 차단기와 차량 인식 장치 등을 통한 주차관리 시스템은 고비용(약 1,100만원)으로, 일부 소규모 주차장 및 간이 주차장을 이를 갖추기 어렵워 사람이 직접 차량을 기록하고 관리하는 비효율적인 방식을 사용하고 있다. 이러한 방식은 인력 낭비와 요금 계산 오류, 차량 위치 혼동 등 다양한 문제를 유발할 수 있다.
   - 기존의 수기 관리 및 단순 차단기 시스템의 비효율성을 해결하고, 관리자에게는 데이터 기반의 매출 분석을, 방문자에게는 실시간 주차 정보를 제공하기 위해 개발

 * **제안 전략**
   
   - 차량 사진을 촬영하거나 갤러리에서 업로드 후 OCR을 활용하여 입차하는 과정으로, 스마트폰 하나로 이뤄지는 통합적인 주차 관리.
   - 주차장 이용자는 자신의 주차 정보를 손쉽게 조회.

* **팀원**

   - 정찬우(BE), 최재우(BE), 최용훈(FE), 이재유(FE), 장재혁(FE)

## 4. 주요 기능

### 관리자 

#### [회원가입/로그인]

* 계정 정보와 주차장 정보를 기입하고 전화번호 인증을 통한 회원가입, Session 방식의 로그인
  
  - ID, PW, PW 재확인, 주차장 이름, 주차장 주소, 주차장 요금 (기본, 분 당), 주차 면 수, 이름, 전화번호
    
* ID 찾기와 PW 재설정 기능
  
  - ID 찾기: 이름을 입력하고 전화번호 인증 시 ID 확인
  - PW 재설정: ID를 입력하고 전화번호 인증 후 PW 재설정

#### [차량 등록]

* 카메라/앨범을 통해 업로드 된 차량 이미지를 OCR을 통해 차량 번호 인식 후 사진에서 GPS 정보와 업로드 시간과 함께 리스트에 등록
  
* 차량 번호를 통한 수동 입차 가능
  
* 차량 등록 시 활용된 이미지를 저장하고, DB에 이미지 경로를 저장, 출차 시 같이 삭제

#### [입차리스트]

* 주차 현황 조회
  
 - 입차 수/주차 가능수, 각 차량 별 차량 번호, 요금, 주차 시간, 입차시간
   
* 주차 차량 관리
  
 - 차량 번호 수정, 블랙리스트 등록, 출차

#### [블랙리스트]

* 블랙리스트 차량 관리 (등록, 조회, 삭제)
  
  - 블랙리스트는 차량 번호와 사유를 입력하여 등록
  - 블랙리스트 차량 OCR 시 등록하기 전 경고

#### [정기권 관리]

* 정기권 차량 관리 (등록, 조회, 환불)
  
  - 정기권은 차량 번호, 기간, 가격을 입력하여 등록
  - 정기권 환불 시 환불 금액 입력
  - 정기권에 등록 된 차량은 입차 시 입차리스트에 요금 0원으로 표기
  - 스케줄링을 통해 기간이 지난 정기권은 자동으로 삭제
    
#### [주차 이력 조회]

* 주차 기록 조회
  
  - 시작일과 종료일을 설정하면 해당 기간 내의 입·출차 기록을 조회 (차량번호, 입·출차 일시, 요금, 주차 시간)
  - 입차일과 출차일 두 개를 기준으로 조회 가능, 차량번호(선택)을 통해 조회 가능
  
* 정기권 기록 조회
  
  - 시작일과 종료일을 설정하면 해당 기간 내의 정기권 기록을 조회 (차량번호, 기간, 금액, (환불 시 - 환불 금액, 환불 일시))
  - 차량번호(선택)을 통해 조회 가능

#### [매출·통계]

* 일 별/기간 별 통계 정보 조회
  
  - 총 매출, 일반 매출, 정기권 매출, 정기권 환불 금액, 입차 수, 출차 수
  - 입·출차 기록과 정기권 기록을 통해 매출·통계 산정

### 방문자

#### [주차 정보 조회]

* 방문자의 차량 번호를 입력하여 주차 정보 조회
  
  - 차량번호, 차량 위치(주소, 지도 매핑), 요금, 주차 시간, 입차 시간, 관리자 연락처

* 길안내 시 네이버지도, 카카오맵 연결되어 방문자의 현 위치에서 차량 위치로 경로 안내


#### [주변 주차장·번호판 미 인식 차량 조회]

* 차량 번호 입력 시 차량 정보가 뜨지 않을 시 (OCR 오인식, 미인식) 현 위치를 기준으로 가까운 주차장 목록을 조회
  
* 목록에서 본인이 주차한 주차장을 선택, 해당 주차장의 미 인식 차량 목록에서 본인의 차량을 선택하여 주차 정보 확인
  - 차량 등록에 이용된 이미지로 본인 차량 확인

#### [입차 요청]

* 미 인식 차량 목록에도 본인의 차량이 없을 시 차량 번호를 입력하여 해당 주차장 관리자에게 입차 요청
